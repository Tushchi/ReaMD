# ReaMD - Markdown Viewer for REAPER

Dockable markdown viewer for audio production workflows.
Link text fragments to timeline items, use teleprompter mode for VO,
and format text with AI.

## Key Features

- **Scenario Linking** - Connect text to timeline items
- **Teleprompter** - Auto-scrolling VO display
- **AI Parse** - Convert messy text to clean markdown (Claude API)
- **Full Markdown** - Headers, lists, tables, code blocks

## Requirements

- REAPER 7.0+
- ReaImGui 0.10+
